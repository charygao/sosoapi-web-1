# sosoapi-web

####关于SosoApi:  
专注于API接口文档管理及线上线下测试的API服务在线工具。  
SosoApi网站：www.sosoapi.com  
论坛：bbs.sosoapi.com  
qq群：305629848  

###使用方法
参考doc目录下的部署文档

### 注意事项
1. 部署项目时不要放在根路径“/”下，访问路径示例：http://localhost:8000/sosoapi/
2. 将pom.xml里的私服地址改成自己的私服地址
3. 该工程依赖sosoapi-base项目，所以先下载sosoapi-base并编译
